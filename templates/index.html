IPL API

This is an API for ipl matches dataset

  • API-1: All teams name (/api/teams)

  • API-2: Team Vs Team record (/api/teamVsTeam?team1='team1_name'&team2='team2_name')

  • API-3: Overall Team record (/api/team-record-overall?team='team_name')

  • API-4: Team record against each team (/api/team-record-against-each-team?team='team_name')

  • API-5: Overall batter record (/api/batter-record?batter='batter_name')
